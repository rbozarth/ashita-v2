Ashita 2

Latest Alpha Release(s)

What's new in v2?

Full Rewrite

Ashita has undergone an entire rewrite from the ground up. This is to help make the code more readable as well as cleanup a lot of things that could have been coded better originally. The base of Ashita v2 is now in a much more maintainable state.

Lua at the core!

With the popular success of LuaCast, we have built Lua into the core of Ashita! This now includes the use of Lua for configuration data as well as the use of Lua addons which can mimic plugins almost entirely! Best of all, nearly all of the ADK is exposed to Lua so you have endless possibilities in creating extensive addons!

Redesigned Plugin Interface (ADK)

Given that Ashita is an actively developed project, we saw many increments to the interface version which required constant recompiling of extensions. With Ashita v2, we have redesigned the plugin interface to use a series of Get/Set functions instead of having plugins rely directly on the data structures. This allows us to update the structures internally while plugins can enjoy not having to be recompiled every update!

Moveable Font Objects

No more are the days of having to constantly edit a configuration for a plugin to set where its text renders to on screen. With Ashita v2, you can now freely move font objects with your mouse [while holding shift] and place them where you want! Giving users the ultimate experience in customizing their interface.
