+ Expand Information
ADK Interface Overhaul This update includes a semi-massive overhaul to the ADK interface. Ashita::AS_String is no longer used as param types. Instead everything is hard-coded to be const char*. This change was done to reduce the conversion cost between the various types. There is a lot of overhead when just using a simple string like "Hello" and passing it as a param that required AS_String. Instead, any needed conversion will be done inside the hook core now instead of between the interface calls. WindowerInput WindowerInput plugin has been introduced in this version. Third-party applications should now work with Ashita v2.

+ Expand Information
Keybind Information - /bind [mods][key] [up/down] [command] - /unbind [mods][key] [up/doown] - /bind list - to list all current keybinds. - /unbind all - to unbind all keys. - /unbindall and /clearbinds - to unbind all keys. - /keyboard_bind same as /bind - /keyboard_unbind same as /unbind Keybinds can use the following mods: ! = alt ^ = ctrl @ = windows key # = apps key + = shift For example: /bind !1 /wave - would bind Alt+1 to /wave when the keys are pressed down. /bind ^1 /wave - would bind Ctrl+1 to /wave when the keys are pressed down. /bind !1 up /wave - would bind Alt+1 to /wave when the keys are released. Memory Function Information - mem:GetBaseAddress(modname) - Obtains the base address of the given module name. - mem:UnprotectMemory(addr, size) - Unprotects the region of memory starting at the address. - mem:FindPattern(modname, bytetable, size, mask) - Finds a pattern of data within the given module. - mem:ReadUChar(addr) - Read an unsigned char. - mem:ReadChar(addr) - Reads a signed char. - mem:ReadUShort(addr) - Reads an unsigned short. - mem:ReadShort(addr) - Reads a signed short. - mem:ReadULong(addr) - Reads an unsigned long. - mem:ReadLong(addr) - Reads a signed long. - mem:ReadULonglong(addr) - Reads an unsigned long long. - mem:ReadLonglong(addr) - Reads a signed long long. - mem:ReadFloat(addr) - Reads a float. - mem:ReadDouble(addr) - Reads a double. - mem:ReadArray(addr, size) - Reads an array. - mem:ReadString(addr, size) - Reads a string. - mem:WriteUChar(addr, val) - Writes an unsigned char. - mem:WriteChar(addr, val) - Writes a signed char. - mem:WriteUShort(addr, val) - Writes an unsigned short. - mem:WriteShort(addr, val) - Writes a signed short. - mem:WriteULong(addr, val) - Writes an unsigned long. - mem:WriteLong(addr, val) - Writes a signed long. - mem:WriteULonglong(addr, val) - Writes an unsigned long long. - mem:WriteLonglong(addr, val) - Writes a signed long long. - mem:WriteFloat(addr, val) - Writes a float. - mem:WriteDouble(addr, val) - Writes a double. - mem:WriteArray(addr, val) - Writes an array. - mem:WriteString(addr, val) - Writes a string.
